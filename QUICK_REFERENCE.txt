## 🧪 Quick Tests

### Check Everything
```bash
python verify_framework.py
```

### Test a Model
```python
from framework.llm_model_tester import LLMModelTester
import asyncio

async def test():
    tester = LLMModelTester()
    result = await tester.test_single_model(
        "llama3.2:latest",
        "What is AI?"
    )
    print(result)

asyncio.run(test())
```

### List All Agents
```python
from framework.agent_registry import get_registry

registry = get_registry()
agents = registry.get_all_metadata()
for agent in agents:
    print(f"{agent.name}: {agent.description}")
```

## 🐛 Troubleshooting

| Problem | Solution |
|---------|----------|
| "Could not connect to Ollama" | Run `ollama serve` |
| "No models found" | Run `ollama pull llama3.2:latest` |
| "Dashboard won't open" | Check port 8501 is free |
| "MCP tool failed" | Check npx/npm installed |

## 💻 Code Snippets

### Get Ollama Client
```python
from framework.ollama_helper import create_ollama_client
client = create_ollama_client()
```

### Create Agent
```python
from framework.agentFactory import AgentFactory
factory = AgentFactory(client)
agent = factory.create_api_agent()
```

### Test Models
